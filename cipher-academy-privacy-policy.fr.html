Politique de confidentialité

Cipher Academy est conçu pour respecter votre vie privée. Cette politique explique comment l'application gère vos données.

Collecte de données

  • Aucune donnée personnelle n'est collectée
  • Aucune donnée n'est stockée sur des serveurs externes ni transmise sur le réseau
  • Aucun outil de tracking ou d'analyse n'est utilisé
  • Aucune connexion internet n'est requise pour jouer

Toutes les données de jeu (progression, paramètres) sont stockées entièrement sur votre appareil via le stockage local. Nous n'y avons pas accès.

Autorisations

L'application ne nécessite que trois autorisations :

  • Internet : nécessaire uniquement pour les achats intégrés via Google Play
  • Paramètres audio : utilisé pour jouer les sons du code Morse via le haut-parleur
  • Facturation : nécessaire pour traiter l'achat unique de la version complète

L'application n'accède pas à votre microphone, caméra, contacts, localisation, ni à aucune autre fonctionnalité sensible de votre appareil.

Achats intégrés

Si vous choisissez d'acheter la version complète, la transaction est gérée entièrement par la plateforme (Google Play ou Apple App Store). Nous n'avons pas accès à vos informations de paiement.

Confidentialité des enfants

L'application ne collecte sciemment aucune information de quiconque, y compris des enfants. Il n'y a aucun compte, aucun formulaire, et aucune requête réseau.

Services tiers

L'application n'intègre aucun SDK tiers pour l'analyse, la publicité ou le tracking. La seule interaction tierce est le service de facturation Google Play pour les achats intégrés.

Modifications de cette politique

Si cette politique est mise à jour, les changements seront reflétés sur cette page avec une date actualisée.

Contact
